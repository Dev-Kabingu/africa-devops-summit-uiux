# Africa DevOps Summit UI/UX

This repository showcases the UI/UX design work for the Africa DevOps Summit, including:

- Landing Page – Clean, engaging entry point with branded visuals and clear calls-to-action.  
- About Page – Minimal design aesthetic presenting the summit’s mission and vision with clarity.  
- Sponsors Page – Grid-based layout highlighting sponsor logos with consistent spacing and visual harmony.  

## Highlights
- Focus on minimal, branded design aesthetics.  
- Emphasis on usability, readability, and authentic storytelling.  
- Designed with Figma and frontend implementation support (HTML, CSS, React).  

## Purpose
To demonstrate professional UI/UX design contributions for a large-scale community event, aligning visuals with branding and enhancing user engagement.
